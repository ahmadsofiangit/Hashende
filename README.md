# Hashende
Hashende is a decode and encode tool made with a python library named "hashlib" able to decode any type of hash from MD5, SHA1, SHA224, SHA256, etc. using a custom wordlist.

### Installation
``` shell
git clone https://github.com/ahmadsofiangit/Hashende.git
```
### Usage
``` shell
cd Hashende
python3 hashdecoder.py [hashed file] [wordlist file]
```

### Example
```
python3 hashdecoder.py hash.txt wordlist.txt
```
